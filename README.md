# 无人机空气动力学实验室 drone-aero-lab

> 用 Three.js 交互式学习无人机的**结构组成**与**空气动力学** —— 纯前端、教学示意级。

**在线体验：<https://gm-aaa.github.io/drone-aero-lab/>**

![](https://img.shields.io/badge/three.js-Z--up-blue) ![](https://img.shields.io/badge/%E7%BA%AF%E5%89%8D%E7%AB%AF-%E9%9D%99%E6%80%81-green) ![](https://img.shields.io/badge/%E6%95%99%E5%AD%A6-%E7%A4%BA%E6%84%8F%E7%BA%A7-orange)

## 这是什么

一个面向无人机初学者的可视化教学项目。已实现**多旋翼**（四/六/八轴）、**直升机**
（单旋翼带尾桨/共轴双旋翼——反扭矩失衡时机身真实自旋、周期变距、自转下滑）与
**垂起固定翼**（倾转旋翼/升力＋巡航——悬停、过渡、固定翼巡航）。

### 结构学习

- 程序化生成的多旋翼 3D 模型：机架中心板、机臂、无刷电机、电调 ESC、螺旋桨（正/反桨交替）、飞控 FC、电池、GPS、脚架
- 点击 3D 部件或侧栏列表，双向高亮并显示部件作用说明
- 螺旋桨按转速真实旋转（正/反桨方向相反）；桨叶带真实**扭转（washout）**：根部迎角大、尖部小，两半绕桨毂 180° 点对称
- 两种垂起固定翼程序化模型：倾转短舱、旋翼与尾流沿枢轴同步转向；升力＋巡航构型具有四副独立升力桨和一副巡航推进桨

### 空气动力学（教学示意级）

| 可调参数 | 实时效果 |
|---|---|
| 桨叶迎角 α（0.75R 参考站） | 桨叶实际倾转；CL 线性段 + 15° 失速骤降 |
| 转速 / 旋翼直径 | 升力随线速度² 与桨盘面积响应；桨叶模型实时变化 |
| 风速 / 风向 / 垂直气流 | 三维风矢量；机身倾斜抗风；上升/下沉气流直接增减升力 |
| 材料（碳纤维/铝/塑料/木） | 改变重量与外观，展示适用场景 |
| 倾转角 / 动力交接 | 分别模拟倾转旋翼的推力转向、升力＋巡航构型的动力分配 |
| 真空速 / 机翼迎角 | 机翼升力随空速²建立；显示失速速度、机翼接管比例与过渡包线风险 |

- **每旋翼升力箭头 + 总升力箭头**，长度与颜色随升重裕度**连续**变化（绿→橙→红）
- **净升力起飞判定**：总升力·cos(抗风倾角) + 垂直气流力 vs 重量 → 爬升 / 悬停 / 升力不足
- **动态阻力**：CD(α) = CD₀ + k·CL²，实时显示气动阻力、风阻与**升阻比 L/D**
- **空气动力学曲线图**：CL(α)、L/D(α)，标注**最大升力**与**最大升阻比**工作点 + 当前 α 游标
- **桨叶剖面教学图**：根/尖两站翼型、相对来流与升力矢量
- **垂起固定翼过渡图**：并列显示垂直起降、当前过渡、固定翼巡航，分解旋翼垂直分量、机翼升力与推进力
- **过渡安全提示**：识别垂直升力缺口、空速不足和机翼失速，避免把“倾转角”误当成“自动获得空速”
- 下洗气流流线、Z-up 工程坐标系（X-Y 水平、Z↑ 竖直 = 升力方向）

> ⚠️ 所有数值为**教学示意**（简化解析模型，非 CFD、非精确工程值），用于建立直觉而非工程设计。

## 本地开发

```bash
npm install
npm run dev     # http://localhost:5173/drone-aero-lab/
npm test        # 空气动力学纯函数单测（Vitest）
npm run build   # 产物在 dist/
```

技术栈：Vite · Three.js（程序化建模，无外部模型资源）· 原生 ES 模块 · Vitest。

## 项目结构

```
src/
├── data/drones.js      # 机型/子类/部件的声明式定义（唯一事实源）
├── builder/builder.js  # 数据 → 带标注 3D 模型；桨叶扭转几何
├── aero/aero.js        # 空气动力学纯函数（全部单测覆盖）
├── scene/scene.js      # Z-up 场景/相机/渲染循环
├── viz/                # 升力箭头/下洗流线/坐标轴/剖面图/曲线图
├── ui/ui.js            # 控制面板/读数/图例/部件列表
└── main.js             # 状态与装配
```

机型结构集中在 `data/drones.js`，气动计算保持纯函数并由 Vitest 覆盖；各类别可按需接入专属可视化。

## 部署

推送到 `main` 即由 GitHub Actions 自动构建并发布到 GitHub Pages
（仓库 Settings → Pages → Source 选 **GitHub Actions**）。
